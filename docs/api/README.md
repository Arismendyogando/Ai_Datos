
# API Documentation

## File Processing Endpoints
- POST /api/upload/drive - Google Drive file upload
- POST /api/upload/local - Desktop file upload
- GET /api/templates - Data templates
- POST /api/process - File processing
- GET /api/export - Data export options

## Supported Formats
- XLS, XLSX
- TXT
- PDF 
- JPG, PNG
